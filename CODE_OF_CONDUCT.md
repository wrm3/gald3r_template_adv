# Code of Conduct

## Scope

gald3r is a solo-maintained project. There is no public contributor community at this time.

This document describes expected behavior for anyone who interacts with the project through GitHub Issues, feature requests, or any official project channel.

## Expected Behavior

All interactions with this project should be:

- Respectful and professional in tone
- Focused on the technical subject matter
- Constructive — report issues clearly and in good faith

## Unacceptable Behavior

The following are not welcome in any project channel:

- Harassment, discrimination, or personal attacks of any kind
- Publishing private information about any individual without consent
- Deliberate disruption of project communications or issue tracking
- Any conduct that would be considered unprofessional in a business context

## Reporting

To report a violation, contact the project maintainer directly via the email address on the [GitHub profile](https://github.com/wrm3). Reports will be reviewed promptly.

## Enforcement

The project maintainer reserves the right to remove, edit, or reject any interaction (issues, comments, other contributions) that violates this policy, and to block users who engage in persistent violations.
